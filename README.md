## Featured Project

### .NET CSV Upload Scheduler

Python未導入のWindows端末でも利用できる、CSV予約アップロードシステムを開発しました。

従来はPython実行環境が必要でしたが、.NET化により配布しやすくし、担当者依存の運用を改善しました。

**主な機能**

- CSVファイル登録
- 指定時刻アップロード予約
- YAML設定管理
- GUI操作対応
- エラー確認運用

**業務上の価値**

- 夜間更新作業の待機削減
- 属人化回避
- 非エンジニア環境でも利用可能
- 引き継ぎしやすい運用設計
